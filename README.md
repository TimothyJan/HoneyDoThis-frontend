# HoneyDoThis-frontend
A relationship preserving Full Stack Web Application for domestic tasks/duties. Frontend portion.

Features:
<ul>
  <li>Create, complete, and delete tasks.</li>
  <li>Filter through All, Active and Completed tasks.</li>
  <li>With the "View All" filter, drag tasks to reorder.</li>
  <li>With the "Active" filter, view all active tasks.</li>
  <li>With the "Completed" filter, clear all completed tasks.</li>
</ul>
